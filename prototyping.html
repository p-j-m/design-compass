프로토타이핑
모바일이나 웹 인터페이스 스크린을 디자인할 때 실제로 동작하는 것이 도움이 될 것입니다. 스케치 프로토타이핑 기능으로 쉽게 인터랙티브 워크플로우를 만들고 유저가 보게 될 디자인을 미리 확인할 수 있습니다.
프로토타이핑이 제대로 작동하기 위해서 문서 내에 아트보드가 있어야합니다. Links and Hotspot Layer를 이용해 아트보드를 연결하여 인터랙티브가 가능한 프로토타입을 만들 수 있습니다. 또한 다른 구성원들에게 공유할 때 Start Points 를 이용해 프로토타입이 어느 페이지에서 시작하는지 설정할 수 있습니다.

프로토타입을 만들면 스케치에서 바로 볼 수 있으며 Sketch Mirror에서 테스트 할 수 있습니다. Sketch Cloud를 이용해 클라이언트나 구성원에게 공유할 수 있습니다.

직접 만들어 보는 것을 좋아하신다면, 프로토타이핑이 어떻게 작동하는지 이해하는데 도움이 되는 Template을 사용할 수 있습니다. 첫 Welcome 창의 Templates에서 열 수 있습니
Links
Links는 문서 내의 아트보드를 연결합니다. 아트보드를 선택한 뒤 연결하려는 아트보드를 클릭하면 연결됩니다.
아트보드 내에 있는 레이어에 Link를 추가할 수 있습니다. Link를 만드려면 Inspector 내부에 있는 Prototyping 옆의 + 버튼을 누르거나 w 키를 누르세요.

아직 연결되지 않은 링크가 마우스 커서를 따라다니고 아트보드에 커서를 올리면 아트보드가 하이라이트됩니다. 연결하려는 아트보드를 클릭하면 연결됩니다.

프로토타이핑을 Preview 창에서 확인할 때 링크가 설정된 레이어를 클릭하면 설정한 아트보드로 이동합니다.

레이어와 아트보드를 연결시키면 Inspector에서 화면이 전환 애니메이션을 설정할 수 있습니다.

Hostspots
Link만으로는 부족한 상황이 있을 수 있습니다. 예를 들어 아이콘이나 메뉴를 클릭하는 영역을 레이어보다 크게 설정하고 싶을 때입니다. Hotspots는 아트보드내에 영역을 설정하여 아트보드에 연결합니다.
관계를 더 쉽게 이해하려면, exportable layers와 slices의 관계와 비슷하다고 생각하면 됩니다.

Hotspot은 Insert 메뉴에서 추가하거나 H를 눌러 그릴 수 있습니다. 레이어를 선택해 Inspector에서 "Create Hotspot"아이콘을 눌러 Link를 Hotspot으로 전환할 수 있습니다.

Link 처럼 Hotspot을 선택하면 Inspector에서 대상 아트보드와 애니메이션 종류를 정할 수 있습니다.

Hotspots은 특히 레이어가 Symbols에 속했을 때 아트보드가 override되는 상황, 즉 Symbol을 재사용하지만 연결되는 아트보드가 바뀌는 경우에 유용합니다. Link로는 불가능하지만 Symbol 내에 Hotspot을 설정 Override에서 "None"을 선택해 Hotspot을 제외할 수 있습니다.

Preferences > Canvas 탭에서 Links와 Hotspot의 색상을 변경할 수 있습니다. 복잡한 프로토타입을 작업할 때 캔버스가 정신없어 보일 수 있습니다. Hotspot을 보이거나 가리고 싶으면 View > Show Prototyping을 선택하거나 Control-F 를 누르면 됩니다.
Start Points
Start Points는 프로토타입에서 특정 아트보드를 시작하는 화면으로 설정하는 마커입니다. Sketch Cloud에 업로드해 다른 사람들에게 공유할 때 어디서부터 시작하는지 알려주고 싶을 때 유용합니다.

Start Points는Preview 창의 아트보드 옆의 깃발 아이콘으로 설정할 수 있습니다. Start Points로 설정되면 깃발 아이콘이 파란색으로 표시됩니다.또한 드롭다운을 눌러 깃발 아이콘이 있는 아트보드를 확인해 어느 아트보드가 Start Points로 설정되어 있는지 확인할 수 있습니다.
Start Point를 지우려면 드롭다운 메뉴를 선택한 뒤 깃발 아이콘을 다시 누르면 아이콘이 회색이 되며 삭제됩니다.

플로우의 앞 부분에 Start Point를 설정하는 것이 좋지만 여러 Start Points를 설정할 수도 있습니다. 여러 아트보드가 있는 복잡한 프로토타입을 제작할 때나 프로토타입의 특정 부분만 공유하고 싶을 때 유용합니다.
Previewing
다른 사람들에에게 프로토타입을 공유할 수 있는 Sketch Cloud를 통해 스케치에서 프로토타입을 바로 확인할 수 있으며, Sketch Mirror 앱으로 iOS 기기에서 확인할 수 도 있습니다.

스케치에서 프로토타입을 시작하기 위해 툴바에서 Preview 버튼을 누르면 Preview 창이 뜹니다. Start Point를 설정했다면 해당 화면에서 부터 미리보기가 시작됩니다. 설정하지 않았다면 현재 선택한 아트보드부터 시작합니다.
Links와 Hotspot 레이어를 추가한 경우 Preview 창을 클릭하며 탭 할 수 있는 영역이 표시되며 클릭하면 해당 아트보드로 이동합니다. Preview 창의 상단에는 뒤로가기 버튼과 모든 아트보드의 리스트를 볼 수 있는 드롭다운 메뉴가 있습니다. 뒤로가기 버튼은 이전 아트보드로 이동하며 드롭다운 메뉴에서 특정 아트보드로 이동할 수 있습니다.

이전 섹션에서 다뤘듯이 Preview 창에서 Start Point 아이콘을 클릭해 Start Points를 삭제할 수 있습니다.

프로토타입 재생을 멈추려면 Preview 창을 닫으면 됩니다.
Sketch Cloud
스케치 내에서 미리보는 것 뿐만 아니라 링크를 받은 사람이 재생할 수 있는 Sketch Cloud에 공유할 수도 있습니다.

Sketch Cloud에 업로드하면 어떤 스케치 문서라도 동일하게 업로드됩니다. 툴바에서 Sketch Cloud 아이콘을 클릭한 뒤 Upload Document를 누르면 됩니다. 더 자세히 알고 싶다면 Sketch Cloud 섹션의 문서에서 알아볼 수 있습니다.

문서에 Start Point를 설정했다면 업로드 후 문서 overview에서 Prototypes 섹션을 볼 수 있습니다.
Prototypes 중 하나를 클릭하면 브라우저에서 특정 Start Points부터 보여집니다. 스케치 내 Preview처럼 Sketch Cloud에서 아트보드를 클릭하면 탭 대상이 표시되며 어떤 영역이 클릭할 수 있는지 보입니다.

문서에 코멘트를 달 수 있게 설정하면 하단 오른쪽 구석에 보는 사람이 각 아트보드에 피드백을 남길 수 있는 코멘트 바가 보입니다. 미리보기 하단 왼쪽의 뒤로가기 버튼을 누르면 현재 아트보드의 정적인 화면으로 이동합니다.

Sketch Cloud에서 프로토타입을 볼 때, 브라우저의 뒤로가기 버튼을 눌러 이전 화면으로 이동할 수 있습니다.
Sketch Mirror
모바일 디자인을 한다면 Wi-fi나 USB를 통해 Sketch Mirror 앱에 연결해 iOS 디바이스에실제 사용자가 경험할 디자인 맥락을 테스트해 볼 수 있습니다.

간단하게 Mirror 앱을 키고 상단 오른쪽의 알림을 클릭해 연결을 확인합니다.
연결되면 기기에서 모든 아트보드를 볼 수 있습니다. 아트보드를 탭하면 설정한 탭 대상을 터치해 다른 아트보드로 이동할 수 있습니다. 아래로 쓸기 제스쳐로 overview로 돌아갈 수 있습니다.
Prototyping and Artboard Presets
프로토타입을 볼 때 아트보드의 크기를 확인하는 것이 중요합니다.
문서에서 커스텀 아트보드를 사용했다면 스케치에서 미리보기 시 아트보드가 높이에 맞춰 축소될 것입니다.
스케치가 스크린 사이즈를 계산하기 위해서 Artboard presets에서 하나를 선택해야합니다.

모바일 혹 웹을 디자인한다면 Preset에서 아트보드를 고른 뒤 스크롤할 수 있도록 높이를 조절해보십시오. Inspector의 아트보드의 이름이 Resized로 바뀌는 것을 확인할 수 있습니다.

Resized가 포함된 아트보드를 재생하면 Preview 창이 스크롤 가능한 화면으로 보여질 것입니다.

커스텀 크기로 아트보드를 사용하면서 정상적인 Preview를 보고 싶다면 Custom Preset을 추가할 수 있습니다.

새 custom preset을 추가하려면 Insert > Artboard (혹은 단축키 A) 를 선택합니다. Create Custom Size를 클릭하고 Inspector에서 이름과 수치를 확인합니다.

커스텀한 아트보드로 예상한 화면이 표시될 것입니다. 아트보드의 높이를 늘리면 스크롤이 가능해지며 Inspector에서 아트보드 Preset의 이름 뒤에 Resized가 붙는 것을 확인할 수 있습니다.
